# adc_pa

Модуль управления АЦП для контроля КСВ на выходе усилителя мощности.

## Структура репозитория

- `rtl/adc_pa.v` – основной модуль на Verilog
- `sim/verilog/tb_adc_pa.v` – тестбенч для симуляции
- `sim/do_files/vsim.do` – скрипт для запуска симуляции в QuestaSim
- `sim/vsim_1_wave.do` – настройки окна волн
- `README.md` – этот файл

## Функциональность

Модуль `adc_pa` реализует:

- Интерфейс AXI‑Lite для чтения/записи регистров управления и результата
- Управление АЦП через SPI (сигналы `adc_sck_o`, `adc_conv_o`, `adc_sdo_i`)
- Регистры: `control_reg` (управление), `rezult_reg` (результат преобразования)

## Порты модуля

| Сигнал          | Направление | Ширина | Описание |
|-----------------|-------------|--------|----------|
| `clk_120_i`     | input       | 1      | Тактовая частота 120 МГц |
| `rst_i`         | input       | 1      | Асинхронный сброс |
| `axi_en_i`      | input       | 1      | Строб AXI‑Lite |
| `axi_data_i`    | input       | 32     | Данные для записи |
| `axi_we_i`      | input       | 1      | Разрешение записи |
| `axi_addr_i`    | input       | 4      | Адрес регистра |
| `axi_vd_o`      | output      | 1      | Строб готовности данных |
| `axi_data_o`    | output      | 32     | Данные чтения |
| `axi_irq_o`     | output      | 1      | Прерывание |
| `tx_active_i`   | input       | 1      | Признак активности передатчика |
| `tx_mode_i`     | input       | 1      | Режим передатчика |
| `adc_sck_o`     | output      | 1      | Тактовая линия SPI |
| `adc_conv_o`    | output      | 1      | Сигнал преобразования АЦП |
| `adc_sdo_i`     | input       | 1      | Вход данных АЦП |
